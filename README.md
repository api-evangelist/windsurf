# Windsurf (windsurf)

Windsurf (formerly Codeium) is the AI-native code editor and plugin family from the Codeium team, featuring Cascade - an agentic chatbot that collaborates with the user across files. Built on a VS Code base with proprietary inference, MCP server support, Devin Cloud sessions, and the SWE-1.5 fast agent model.

## APIs
- **Windsurf Editor** — Standalone AI-native IDE (VS Code fork) with Cascade, autocomplete, chat, command, MCP, Devin Cloud sessions, SWE-1.5 fast agent.
- **Windsurf Plugins** — VS Code / JetBrains / Vim / Emacs / Visual Studio / Sublime Text plugins.
- **Windsurf Enterprise (Codeium for Enterprise)** — Self-hosted enterprise SKU with admin, SSO, analytics, RBAC.

### Plans (May 2026)
- **Free** — $0/mo: standard usage allowance.
- **Light** — Daily/weekly refreshing usage allowance.
- **Pro** — $20/mo: standard usage tier; API pricing for overage.
- **Max** — $200/mo: heavy usage; API-priced overages.
- **Teams** — $40/user/mo: centralized billing, admin dashboards.
- **Enterprise** — Custom: SSO, RBAC, advanced analytics, self-host options.
